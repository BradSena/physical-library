# ADR-0004 — Original location is preserved

## Status

Accepted

## Context

When a disc is loaned or displayed elsewhere, the user still needs to know where
it belongs when returned.

## Decision

A physical copy keeps an original/home location distinct from current state or
temporary holder.

## Consequences

Positive:

- easy returns;
- less shelf disorder;
- clear inventory history.

Negative:

- data model is slightly more complex.
